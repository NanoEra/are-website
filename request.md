# 主页设计文档

## 1. 整体页面设计
### 1.1 配色方案
- 可以采用原本的暗紫色->黑色的辉光效果及渐变等，调色可以参考原来的官网
### 1.2 标题栏
- 左侧写"ARE曙光引擎"，副标题"一款由NanoEra自主研发的高性能实时物理渲染引擎"，右边一个首页标签一个下载标签一个资源中心标签，资源中心标签子菜单有开发文档和FAQ以及前往GitHub仓库
- 标题栏可以直接参考原来的官网或者我个人主页
### 1.3 其他
- 主要还是可以先翻阅一下我给你的那几个参考网站：
```
ue官网：https://www.unrealengine.com/zh-CN
godot hub网站：https://godothub.com/
godot引擎官网：https://godotengine.org/zh-cn/
to的个人网站：https://www.ternaryop.top/
隔壁友商网站设计在引用
```

## 2. 首页
- 大主页需要展示渲染引擎logo、英文和中文名称（英文Aurora Rendering Engine，中文是曙光引擎），可以类似ue的官网一样加一句slogan，比如"曦色未名，待君执笔。"之类的
- 还需要堆模块，来个我们的特点、安装指南（你想办法占位然后跳转到使用指南页面即可）、效果一览和底部的面板（底部面板我觉得可以参考原官网的样式，樱老师写的挺好的）

## 3. 下载页面
- 下载部分搞个大标题，给下载链接，一个"前往GitHub Releases"，再给一个下载引擎库的链接，比较简单

## 4. 资源中心
- 四个跳转链接，分别是GitHub仓库，开发文档和常见问题解答

## 5. 文档页面
- 文档页面主要是开发文档，写几个大模块，第一个是快速安装，第二个是示例程序，第三个是API详解，第四个模块是常见问题解答，跳转到FAQ页面

## 6. FAQ页面
- FAQ页面主要是一些常见问题的解答，大概可以参考CFTQ官网(https://cftq-cloud.nanoera.top/support/faq.html)的实现方法（但是布局之类的要适配你设计的风格哦）

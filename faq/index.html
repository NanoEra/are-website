常见问题与使用指南

Aurora Rendering Engine 是什么？

Aurora Rendering Engine (ARE) 是一款由 NanoEra 团队自主研发的高性能实时物理渲染引擎。曦色未名，待君执笔——我们致力于为创作者和开发者提供卓越的渲染体验。

ARE 是免费的吗？

是的，ARE 面向个人用户完全免费。我们致力于为所有用户提供高质量的渲染技术，即使是面向开发者，也有免费的开源方案。

支持哪些平台？

目前支持 Windows 10+/Linux (Ubuntu 20.04+)。移动端支持正在开发中。源代码完全开源，可在 GitHub 获取。

如何获取源代码？

访问我们的 GitHub 仓库，可以克隆项目或下载发布版本。遵循 MIT 许可证。

如何参与开发？

我们欢迎社区贡献！请阅读 GitHub 上的 CONTRIBUTING 指南，提交 Pull Request 或报告 Issue。

有哪些技术文档？

完整的 API 文档、示例代码和架构说明可在 GitHub 仓库的 docs/ 目录中找到。我们也在不断完善中文文档。

遇到问题如何获取帮助？

可以通过 GitHub Issues 提交问题，或加入我们的社区讨论。对于商业支持需求，请通过官网联系方式咨询。

如何联系团队？

发送邮件至团队邮箱，或在 GitHub 上提交 Issue。我们会尽快回复。

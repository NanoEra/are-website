一般问题

CFTQ Cloud 是什么？

CFTQ Cloud 是一款基于云的安全解决方案，提供文件扫描、威胁检测和安全分析服务。我们采用先进的启发式引擎和机器学习技术，为用户提供快速、准确的安全防护。

CFTQ Cloud 是免费的吗？

是的，CFTQ Cloud 面向个人用户完全免费。我们致力于为所有用户提供高质量的安全服务，即使是面向开发者，也有免费的API接口方案。

CFTQ Cloud 与其他杀毒软件有什么区别？

CFTQ Cloud 采用自研的新一代启发式反病毒引擎，结合由纳米时代团队自研的关联数据挖掘算法与高维关联度分级筛选算法的训练引擎，在保障查杀率的同时实现了高速的查杀速度。同时，我们的数据库内循环系统能够自动分析上传的文件并实时提取特征，保障了对新型病毒的查杀能力。

安全与隐私

上传的文件会如何处理？

上传的文件仅用于安全分析，扫描后立即从服务器删除，仅保留必要的元数据以改进检测能力。我们严格遵循隐私保护原则，确保您的文件安全。

我的数据安全吗？

是的，我们严格遵循"三不"原则（不存储、不泄露、不滥用），服务器与网页均为自有基础设施，确保数据安全。所有数据传输都经过加密处理。

CFTQ Cloud 如何保护用户隐私？

我们采用端到端加密技术保护数据传输，严格的数据访问控制，定期安全审计，以及透明的隐私政策。您可以查看我们的隐私政策及服务协议了解更多详细信息。

客户端使用

如何下载和安装客户端？

前往下载页面获取 Windows 版安装包，按提示安装即可。我们提供离线CLI扫描工具、GUI扫描器和完整杀毒软件等多种选择。

客户端支持哪些操作系统？

目前客户端仅支持 Windows 7 及以上版本。对于其他操作系统（如 macOS、Linux），您可以通过浏览器使用我们的在线扫描服务，无需安装任何软件。

系统要求是什么？

推荐系统要求：Windows 7 及以上，2GB 内存，240MB 可用磁盘空间，以及互联网连接用于更新病毒库。具体系统要求请参考下载页面。

API 与开发者

如何获取 API 密钥？

您可以访问API 文档页面申请 API 密钥。我们为个人开发者和商业用户提供不同的方案。

API 的调用限制是多少？

免费方案：每天 100 次调用，每月 3000 次调用。如需更高限制，请查看我们的付费方案或联系客服。详细的 API 限制和定价信息请参考API 文档。

支持哪些编程语言调用 API？

我们的 API 支持任何能够发送 HTTP 请求的编程语言。我们提供 Python、JavaScript、Java 等语言的示例代码。具体请参考API 文档中的示例部分。
